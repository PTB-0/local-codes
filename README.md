# local-codes
local kodlar

Bu kodlar ana bilgisiyardaki kodlardır 
Dosyalar şunlara göre ayrılmıştır:
    1 - uygulama dili veya uygulama dosya tipi 
    2 - nitelik (çalışmayanlar bir yere toplanmış olabiir)
    3 - özel tür kodlar (bazı şeyler yaptığı için açık olmayan veya dsoya dizinine uygun olmadığı için ayrı tutulan uygulamalar)
